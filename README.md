# Interface-Figma 
Wireframe mobile de baixa fidelidade no Figma, a partir de uma situação-problema realista, com foco em:  tarefa principal do usuário, jornada de uso, arquitetura da informação, clareza funcional, flexibilidade de uso, validação por teste com usuários.

## Planejador de Eventos do WhatsApp - Design de Recursos

###📌 Sobre o Projeto
Este projeto apresenta o design de uma nova funcionalidade do WhatsApp: a Criação de Listas de Convidados e Eventos. A proposta visa otimizar a organização de eventos diretamente pelas conversas, permitindo que o usuário gerencie participantes de forma prática e visual, sem sair do ecossistema do aplicativo.

###🚀 Fluxo de Experiência (UX)
O fluxo foi desenhado para ser intuitivo, seguindo os padrões de design (Material Design/iOS) já utilizados pela plataforma:

Acesso Facilitado: Integração no menu de anexo (clipe de papel) e no menu de opções (três pontos).

Configuração da Lista: Definição do nome do evento e quantidade de convidados.

Seleção Dinâmica: Interface para escolha de contatos e adição rápida de novos membros à lista já existente.

Gestão Visual: Checklist de membros para controle de quem já foi confirmado ou adicionado.

###🎨 Interface (UI)
As telas foram projetadas com foco em baixa/média fidelidade (wireframes) para validar a usabilidade antes da aplicação da camada visual final.

Principais Telas:
Página Inicial & WhatsApp: Ponto de entrada da funcionalidade.

Guia de Novidade: Tooltip explicativo para o usuário sobre a nova função.

Menu de Anexos: Inclusão do ícone "Criar Lista" junto às opções de Foto, Documento e Localização.

Configurador de Evento: Inputs para nome, número de pessoas e seleção de contatos.

###🛠️ Detalhes de Design
Componentes: Uso de botões arredondados, seletores numéricos (steppers) e listas de contatos padronizadas.

Microinterações: Fluxo de confirmação ao adicionar novos membros ("Adicionado").

Arquitetura de Informação: Priorização da facilidade de acesso, reduzindo o número de cliques para criar um evento.
